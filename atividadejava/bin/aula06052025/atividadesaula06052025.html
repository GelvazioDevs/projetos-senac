1. Faça um algoritmo que tenha uma função que execute a soma de 2 numeros. Os 2 numeros devem ser informados de forma fixa.
  2. Faça um algoritmo que tenha uma função que receba como parametro o nome da pessoa, e retorne a seguinte frase concatenada: "Meu nome é: " + nome.
  3. Faça um algoritmo que tenha uma função que calcule a area de um triangulo, recebendo como parametro, as 3 medidas.
  4. Faça um algoritmo que use a funcao do algoritmo anterior e ao final verifique o tipo do triangulo, isosceles, escaleno ou equilatero.
